How to Use PacaFilmDB

PacaFilmDB is a movie database where you can search for trending movies, popular films, and detailed information about your favorite movies. Start by using the search bar on the Home Page to find movies you're interested in. Each movie card contains information like the movie poster, title, and a description.

Favorite Movies Feature

To save your favorite movies, click the "Add to Favorites" button located on each movie's page. This will store the movie data locally in your browser using localStorage. You can view your saved movies on the Favorites Page. This feature makes it easy to keep track of the movies you want to watch later.
